# Bitcoin

Bitcoin is a decentralized digital currency invented by [[Satoshi Nakamoto]]
in the 2008 whitepaper. It uses [[proof of work]] consensus to secure the
chain without a trusted third party.

Key properties:
- 21M hard supply cap
- UTXO model (distinct from [[Ethereum]]'s account model)
- ~10 minute block time
- Mined via [[SHA-256]] hashing

See also: [[Lightning Network]] for payment scaling.
